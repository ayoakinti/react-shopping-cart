# E-commerce app built using React+TypeScript with dummy data from https://fakestoreapi.com/
## Features
- Authentication pages to enter the platform
- Products page where you find all available products
- Categories tab: where products are divided according to their categories
- Cart: This shows all the items you want to buy as a user
- PayStack gateway: Just for dummy payments
## Build Setup
``` bash
# clone repo
git clone https://github.com/iamayoakinti/react-shopping-cart.git

# install dependencies
npm install

# serve with hot reload at localhost:3000
yarn start

# build for production with minification
yarn build
```
